Công cụ hỗ trợ tải truyện từ hệ thống **Hako** (docln.net, ln.hako.vn, docln.sbs) chuyển đổi trực tiếp thành file **EPUB** chất lượng cao ngay trên trình duyệt.

## ✨ Tính năng nổi bật

- **Chất lượng ảnh gốc (Original Quality):** Không nén ảnh, giữ nguyên độ sắc nét từ máy chủ Hako.
- **Hỗ trợ đa tên miền:** Hoạt động mượt mà trên tất cả các trang dự phòng (docln.net, ln.hako.vn, docln.sbs).
- **Tiếp tục tải (Resume):** Ghi nhớ chương đang tải dở, cho phép tiếp tục nếu lỡ tay làm mới trang.
- **Trang giới thiệu:** Tự động tạo trang bìa, mục lục và tóm tắt nội dung chuyên nghiệp.

## 🚀 Hướng dẫn cài đặt

### 💻 Trên Máy tính (PC/Laptop)
1. Cài đặt tiện ích **[Tampermonkey](https://www.tampermonkey.net/)** (Chrome, Edge, Firefox...).
2. Bấm vào [ĐÂY](https://github.com/youkawa-S0L4/Hako-Epub-Downloader/raw/refs/heads/main/Hako_epub.user.js) để cài đặt Script.
3. Bấm **Install** (Cài đặt) khi bảng điều khiển hiện ra.

### 📱 Trên Điện thoại (Android)
1. Cài đặt trình duyệt **Kiwi Browser** từ CH Play.
2. Cài đặt tiện ích **Tampermonkey** từ Chrome Web Store trên Kiwi.
3. Mở link script và bấm **Install**.

### 🍎 Trên Điện thoại (iOS/iPhone)
1. Cài đặt ứng dụng **Userscripts** từ App Store.
2. Tải file `.user.js` về máy và lưu vào thư mục của ứng dụng Userscripts.
3. Kích hoạt tiện ích trong Safari.

## 🛠 Cách sử dụng

1. Truy cập vào trang chi tiết của bộ truyện bạn muốn tải.
2. Bảng điều khiển sẽ hiện ra ở góc trên bên trái màn hình.
3. Bấm **BẮT ĐẦU CÀO** để bắt đầu quá trình.
4. Chờ thanh tiến trình chạy đến 100%, file `.epub` sẽ tự động tải xuống.
